# Example Extracts

These CSV files are small real-data extracts from processed summary outputs. They document expected columns and data shapes without releasing raw climate archives, generated full panels, or complete result tables.

- `sample_dskill_summary.csv`: model-threshold bias-adjustment skill summary.
- `sample_crop_threshold_dskill.csv`: crop-specific skill differences across EDD thresholds.
- `sample_raw_model_mae.csv`: raw CMIP6 model error summary against ERA5.
- `sample_public_paper_summary.csv`: compact public-paper model-by-threshold summary.

Values are intentionally limited to a few rows and should be treated as documentation examples, not as a complete replication dataset.
